# Airbnb in Switzerland, visualized.

Data Visualization Project for COM-480 2021 at EPFL.

```

├── assets
|   ├── css                <- Styling

|   ├── js                 <- JavaScript code

├── index.html             <- Single page HTML code
├── README.md              <- The current file
```
